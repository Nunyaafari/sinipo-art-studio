## 🛠️ Installation & Setup

### 1. Clone the Repository
```bash
git clone <repository-url>
cd premium-art-studio-website
```

### 2. Frontend Setup
```bash
# Install frontend dependencies
npm install

# Start the frontend development server
npm run dev
```

The frontend will be available at `http://localhost:5173`

### 3. Backend Setup
```bash
# Navigate to backend directory
cd backend

# Install backend dependencies
npm install

# Create environment file
cp .env.example .env
```

### 4. Configure Environment Variables

Edit the `.env` file in the backend directory:

```env
# Paystack Configuration
PAYSTACK_SECRET_KEY=sk_test_your_secret_key_here
PAYSTACK_PUBLIC_KEY=pk_test_your_public_key_here

# Admin CMS Configuration
ADMIN_SECRET_KEY=your_admin_secret_key_here

# JWT Authentication Configuration
JWT_SECRET=your-super-secret-jwt-key-change-in-production
JWT_EXPIRES_IN=7d

# Email Configuration (for verification and password reset)
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=your-email@gmail.com
SMTP_PASS=your-app-password

# Database Configuration (PostgreSQL)
DB_HOST=localhost
DB_PORT=5432
DB_USER=postgres
DB_PASSWORD=your_database_password_here
DB_NAME=sinipo_art

# Cloudinary Configuration (for file uploads)
CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_API_SECRET=your_api_secret

# Server Configuration
PORT=3001
NODE_ENV=development
FRONTEND_URL=http://localhost:5173
```

**To get your Paystack keys:**
1. Sign up at [Paystack](https://paystack.com)
2. Go to Settings → API Keys & Webhooks
3. Copy your Test Secret Key and Test Public Key

**To get your Cloudinary credentials:**
1. Sign up at [Cloudinary](https://cloudinary.com)
2. Go to Dashboard → Account Details
3. Copy your Cloud Name, API Key, and API Secret

**Authentication Setup:**
- Set a secure `JWT_SECRET` for token authentication
- Configure email settings for verification and password reset
- Set a secure `ADMIN_SECRET_KEY` for accessing the admin dashboard

**Admin CMS Access:**
- Access the admin panel at: `http://localhost:5173/admin`
- Use your `ADMIN_SECRET_KEY` to access the dashboard

### 5. Set Up Database
```bash
# Navigate to backend directory
cd backend

# Generate database migrations
npm run db:generate

# Run migrations (creates tables)
npm run db:migrate

# Seed initial data (optional)
npm run db:init
```

### 6. Start the Backend Server
```bash
# Start backend development server
npm run dev
```

The backend API will be available at `http://localhost:3001`

## 💳 Payment Flow

1. **Add to Cart**: User adds artworks to shopping cart
2. **Checkout**: User fills in customer information
3. **Payment Initialize**: Backend creates payment transaction with Paystack
4. **Paystack Redirect**: User is redirected to Paystack payment page
5. **Payment Processing**: User completes payment on Paystack
6. **Callback**: User is redirected back to success/pending page
7. **Verification**: Backend verifies payment status with Paystack
8. **Confirmation**: User sees payment confirmation

1. **Select File**: User selects image file via drag-and-drop or click
2. **Validation**: Check file type and size limits
3. **Optimization**: Compress and resize image using Sharp
4. **CDN Upload**: Upload optimized image to Cloudinary
5. **Variants**: Generate multiple sizes (thumbnail, small, medium, large)
6. **URL Return**: Return Cloudinary URLs for frontend use
7. **Cleanup**: Remove temporary local files
